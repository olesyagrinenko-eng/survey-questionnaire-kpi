Конструктор анкеты под тест креативов

Онлайн-опрос · 18–54 · география Яндекс Лавки · панель ОнИн. Соберите блоки, показатели и выгрузите ТЗ программисту в DOCX.

Сбросить все шаги и собранную анкету? Это нельзя отменить.

1. Проект и этап

Укажите название для шапки спецификации и тип замера: до выхода кампании (претест) или после (посттест).

2. Материалы

Сколько стимулов каждого типа попадёт в анкету (0 — если типа нет). Сценарий и концепция задаются отдельными счётчиками: при > 0 подключаются соответствующие блоки вопросов.

Минимум один тип материала с количеством > 0, иначе собрать анкету нельзя.

Ссылки на стимулы (картинки / превью)

Для каждого номера стимула можно указать URL или загрузить файл — ссылка попадёт в спецификацию и в DOCX для программистов. Видеоролики обычно ведут на хостинг; для макетов — превью-картинка.

3. Типовые показатели

Блоки раскрываются по заголовку. Галочка у группы включает весь блок; внутри можно снять отдельные вопросы. Скрининг обязателен целиком. Если в анкете есть ролики, вопрос про дебрендированный макет автоматически не включается (остаётся один вопрос про дебренд по ролику).

4. Дополнительно

Опциональные блоки (клик-тест, recall и т.д.) — только если есть подходящие материалы. Блок можно включить целиком и при необходимости снять отдельные вопросы внутри (если их несколько).

5. Структура анкеты

Проверьте формулировки и типы ответов. Можно править текст, варианты (через «;»), добавлять строки. Затем выгрузите DOCX для передачи в разработку.
